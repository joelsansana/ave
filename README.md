# Ave - Catholic Habits

A Flutter app to help Catholics build daily faith habits.

## Getting Started

### Prerequisites

- Flutter SDK 3.0+
- Dart 3.0+

### Install Dependencies

```bash
cd catholic_habits
flutter pub get
```

### Run

```bash
flutter run
```

## Features

- [ ] Daily habit tracking with reminders
- [ ] Saint of the Day
- [ ] Liturgical calendar
- [ ] Lectio Divina guide
- [ ] Faith journal with AI insights
- [ ] Family profiles

## Screens

- Home (habits + saint)
- Calendar (streaks)
- Read (daily readings)
- Profile (stats + settings)

## Tech Stack

- Flutter (iOS, Android, Web)
- Firebase (Auth, Firestore)
- OpenAI (journal insights)
